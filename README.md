Приложение меняет цвет экрана и частоту звука в зависимости от напряженности и направления магнитного поля вокруг.
Цвет зависит от направления поля (расположение магнита).
Частота звука зависит величины напряженности (расстояние от магнита до iPhone).

Лучше брать магнит посильнее.
Приложение психоделическое, но при длительной тренировке - можно испольнить простенькую мелодию. :))))
